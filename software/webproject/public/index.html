Εργασία 1: Εγγραφή με AJAX Έλεγχο

Στόχος: Έλεγχος διπλότυπων username/email με AJAX κατά την πληκτρολόγηση

Χαρακτηριστικά:

  • AJAX duplicate checking με debouncing (500ms)
  • Οπτικό feedback (πράσινο ή κόκκινο)
  • Έλεγχος και στους δύο πίνακες (users, bands)
  • XSS protection με sanitization
Φόρμα Εγγραφής Χρήστη Φόρμα Εγγραφής Μπάντας

Εργασία 2: Προφίλ Χρήστη με Session Management

Στόχος: Single Page Application με sessions για προβολή/επεξεργασία προφίλ

Χαρακτηριστικά:

  • Login/Logout με sessions
  • SPA (χωρίς ανανέωση σελίδας)
  • Protected routes με requireAuth middleware
  • Επεξεργασία προφίλ (εκτός από username/email)
Προφίλ Χρήστη (SPA)

Εργασία 4: REST API για Κριτικές

Στόχος: Ανεξάρτητο REST API για κριτικές μπαντών

Endpoints:

  • POST /review/ - Δημιουργία κριτικής
  • GET /reviews/:band_name - Λήψη κριτικών
  • PUT /reviewStatus/:review_id/:status - Ενημέρωση status
  • DELETE /reviewDeletion/:review_id - Διαγραφή κριτικής
Interface test REST API
